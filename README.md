# Port Killer

Windows 端口查找结束工具。输入端口号，快速定位占用该端口的进程并一键结束。

## 功能

- 输入端口号，查找所有占用该端口的进程
- 表格展示：协议、地址、状态、PID、进程名
- 支持单选/多选结束进程
- 右键菜单、双击快速结束
- 自动过滤 PID 0 和重复进程
- 仅显示 LISTENING / ESTABLISHED 状态的连接

## 使用方法

### 直接运行（需要 Python 环境）

双击 `启动.bat`

### 打包版本

| 打包方式 | 文件 | 说明 |
|---------|------|------|
| 单文件 | `dist\PortKiller.exe` | 发布方便，启动略慢 |
| 目录版 | `dist\PortKiller\PortKiller.exe` | 启动更快 |

### 从源码打包

```bash
# onefile 模式（单个 exe）
.\打包exe.bat

# onedir 模式（目录版，启动更快）
.\打包onedir.bat
```

## 操作说明

1. 在顶部输入框输入端口号（如 8080）
2. 按回车或点击「查找」按钮
3. 选中表格中的行（可多选 Ctrl/Shift）
4. 点击「结束选中进程」或双击/右键快速操作
